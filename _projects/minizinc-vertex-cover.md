    <h2>MiniZinc Vertex Cover</h2>

    <p>
        This project models the vertex cover problem using MiniZinc. The goal is
        to choose a small set of vertices so that every edge in a graph is covered
        by at least one selected vertex.
    </p>

    <h3>Focus</h3>

    <ul>
        <li>Representing graph constraints clearly</li>
        <li>Encoding optimization objectives</li>
        <li>Testing small examples before scaling up</li>
    </ul>
