## 感想

以前上中学的时候，网课期间无聊也搞了个基于 GitHub page 搭建的个人网站，但是当时懂得太少，折腾了好半天，也是一团糟，于是这件事就这么搁置了。最近也算是心血来潮，加上对于 GitHub 也更熟悉了，决定再次搭建一个个人主页。就像我说的，to live is to love. 希望再未来，我还能永远记得今时今日的这些美好与感动的瞬间。
