## 2026.04.09

今天主要是回顾了一下 Guido 教授之前发给我的文件，但是这份文件太过简略，导致里面又很多东西我并没有完全弄懂。正当我一筹莫展的时候，我想起了应该可以查找一下 MiniZinc 的[官方文档](https://docs.minizinc.dev/en/stable/index.html)。下面简单写一个总结。

一开始，我在 Guido 教授的文件中先提取出了整个 research 的大致框架：

- 对 MiniZinc 进行 Declarative Semantics 的定义。粗略地说就是去决定 MiniZinc 的句子中哪些东西是对的，哪些东西是错的。如果学过一点逻辑学，其实可以认为是对这个 language 的 sentences 赋 Boolean value. 以上为我个人理解，不保证准确，因为我也刚接触这个概念。
- TBC...(没有通读全文导致的)

于是我就准备先找找 MiniZinc 的官方 declarative semantics. 但是由于我没有接触过这个语言，所以我是从头开始读 Handbook 的，实际上这个 declarative semantics 的部分在 Part4. 

以下为对于 MiniZinc 的简单介绍。MiniZinc 模型常用于求解约束问题和优化问题，但是 MiniZinc 本身并不是求解器，它是通过把 MiniZinc model 和数据集通过 complier 翻译成 FlatZinc 格式，再调用 solver 进行求解。

而不直接使用 FlatZinc 进行建模的原因，是因为 FlatZinc 难以被人类直接阅读，而 MiniZinc 的格式则接近于自然数学语言，所以书写 MiniZinc 代码是更为方便的。

通过数据集，MiniZinc 可以不只求解一种给定数据的情况。一个好的 MiniZinc 模型可以通过输入不同的数据集，去解决多种不同的情况。（对于同类问题）

以下建议参考 Handbook 中的例子，对于理解 MiniZinc 的格式和它具体能解决什么样的问题十分有帮助。比如涂色问题、优化问题（找到生产方式使得利润最大化）、金属表面温度分布问题等等。

今天我就暂时读完了 2.1, 感觉大脑有点 overload 了，需要干点别的事换换脑子。（感觉主要是一直看例子纯输入格式，没有什么思考的部分，确实有点无聊，但是多看一些例子确实能增强对 MiniZinc 适用场景的了解。如果把应用写在 abstract 里面，感觉有点像一个 list, 属于扫一眼就忘了，而且对于数学建模问题类型不熟悉的话，可能也不知道这个 abstract 在说什么）

## 2026.04.10

简单浏览了一下 MiniZinc 的 syntax 定义，它使用的是 EBNF. 但是 Handbook 中并没有介绍 MiniZinc 的 declarative semantics, 或许是这部分工作还没有完成。

感觉应该在 [2.9](https://docs.minizinc.dev/en/stable/flattening.html) 找一下。从 MiniZinc 到 FlatZinc 的 translation 如下：

1. 对于带有 data file 的情况，先把所有的参数替换成具体数据。
2. 通过 subexpression and syntax of FlatZinc, 将 constraint 变为 FlatZinc 格式。
3. MiniZinc 带有简单的 bounds analysis 功能。
4. ...

总的来说这算是一份概述，但是并没有讲 MiniZinc 中是如何用具体的代码去实现的。

接着回顾了一下最开始的 Problem 的文档，简单总结一下问题：

1. 现在的问题是，我不知道他们是否已经完成了 declarative semantic 的定义。
2. 需要 implement（实现）一个 interpreter(解释器) 来实现这个语义，并且这个过程可以被形式化验证。并且对于满足性问题，这个 interpreter 需要能够验证求解器得出的解的确满足题意。（没看懂，什么叫实现一个语义？）
3. 任务目标是证明原始 MiniZinc 模型与翻译后的 FlatZinc 模型在逻辑上是**等价的**。
4. 如果我的理解没有问题，目前阶段是 MiniZinc 已有一套算法去进行翻译这个过程，但是还没有验证这个算法生成的翻译过程是正确的。目前想法是，基于操作语义去构建一个经过完全形式化验证的翻译工具（相当于替代现有的翻译算法）；或者也是开发一种新的翻译工具，但是在输出翻译结果的同时输出翻译过程的证明。（不过这个任务应该和我现在阶段关系不大，因为比较复杂。我估计的话，我现在能做的东西应该和那个 interpreter 有关）
